universal selector

it indicates with *

it selects with all elements

grouping selector

it is used to select group of elements
it is indicated with comma(,)

combined selector

    descendent slector

    it selects all child elements (direct/indirect)

                    parent child{

    child selector

    it slects direct child elements

                    parent > child{

                }
    adjacent sibling selector

    it selects immediate adjacent element

                    main + adjacentelement{
                    //it selects immediate adjacent element

                }
    general sibling selector

    it selects all adjacent elements

                    main ~ adjacentelement{
                    //it selects all siblings to main element

                }
